代码库
购买 MkSaaS 后获得的所有代码库

概述
当您购买 MkSaaS 模板时，订单页面将要求您填写您的 Github 用户名 (不是您的 Github 邮箱，例如个人首页是https://github.com/fox就填入fox)，支付成功之后，您将收到 5 封邮件邀请，进入 5 个私有代码库，每个代码库都针对不同的使用场景而设计，可以根据您的具体需求进行定制。

代码库
1. MkSaaS 模板
仓库: mksaas-template

网址: https://demo.mksaas.com

状态: ✅ 已就绪

部署：https://demo.mksaas.com 使用 main 分支，Neon数据库，部署在 Vercel 上。

https://cf.mksaas.com 使用 cloudflare 分支，Neon数据库，部署在 Cloudflare Worker 上。

https://cf-d1.mksaas.com 使用 cloudflare-d1 分支，Cloudflare D1 数据库，部署在 Cloudflare Worker 上。

MkSaaS 模板是一个完整的 Next.js 模板，可用于构建 AI SaaS 网站，这是所有其他代码库的构建基础。

MkSaaS Template
